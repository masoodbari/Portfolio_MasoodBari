WHAT IS GIS?

A Geographic Information System (GIS) is a computer system that analyzes and displays geographically referenced information. It uses data that is attached to a unique location. WebGIS is an advanced form of Geographic Information Systems available on web platforms. It kicked off as a GIS running in web browsers and has evolved into Web GIS serving desktop and mobile clients. A spatial database is a database that is enhanced to store and access spatial data or data that defines a geometric space. These data are often associated with geographic locations and features, or constructed features like cities.
